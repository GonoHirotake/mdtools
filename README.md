mdtools
==========
#About
scripts for analysis of MD trajectory

using python, ruby, Gromacs-built-in-tools(>=5.0.5) etc...

#Required python package

for using MDAnalysis module, **python 2.x is required**

* MDAnalysis (>=0.13.0)
* numpy (>= 1.10.4)
* seaborn (>= 0.7.0)
* matplotlib (>= 1.5.0)

#Required ruby package

* bioruby(>= 1.5.0)
